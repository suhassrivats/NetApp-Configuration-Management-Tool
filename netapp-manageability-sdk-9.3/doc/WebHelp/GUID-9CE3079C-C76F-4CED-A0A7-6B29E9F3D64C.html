Utilities of the SDK
Purpose
The NetApp Manageability SDK provides utilities that help users test the APIs. This section provides details about the two utilities, apitest and ZExplore.
This section contains
•
apitest is a command-line interface. You can use this to test ONTAP APIs and DataFabric Manager APIs for the OnCommand Unified Manager Core Package.
•
ZExplore Development Interface (ZEDI) is a graphical user interface that can be used to test the ONTAP APIs and DataFabric Manager APIs for the OnCommand Unified Manager Core Package. ZEDI for Windows and Linux are available at zedi/.
Click Help > ZExplore Development Interface User Guide to view the user guide from the ZEDI interface.
ZEDI supports Java Runtime Environment (JRE) 1.6.0 or later.
NetApp offers ONTAP Simulator, which provides the experience of administering a NetApp storage system with most of the features of ONTAP.
All NetApp customers can access the simulator by registering at the Alliance Partner Program.

apitest
apitest is a command-line utility to test APIs. This utility is suitable for API users who are at the beginner's level. This utility is available in C, Java, Perl, C#, VB.NET, PowerShell in Windows, Python, and Ruby. In UNIX-like environments, apitest is available in C, Java, Perl, Python, and Ruby.
