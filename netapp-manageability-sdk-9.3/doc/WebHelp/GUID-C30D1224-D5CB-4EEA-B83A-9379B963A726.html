NetApp Manageability SDK 4.1
The following are new in NetApp Manageability SDK 4.1:
•
Support for the following Data ONTAP releases:
◦
Data ONTAP 7.3.4
◦
Data ONTAP 7.3.5
◦
Data ONTAP 8.0.1 (7-Mode only)
◦
Data ONTAP 8.1 APIs, which includes 7-Mode, Cluster-Mode, and Vserver APIs
•
Support for DataFabric Manager server APIs for OnCommand Core Package 5.0
•
Support for Certificate-Based Authentication for DataFabric Manager server
•
Support for creating PowerShell Cmdlets for Data ONTAP APIs and DataFabric Manager server APIs for OnCommand Core Package
They are available at the NetApp Support Site
•
Support for Python and Ruby language
•
ZExplore support for Data ONTAP 8.1 (both 7-Mode and Cluster-Mode) and DataFabric Manager server APIs for OnCommand Core Package. It includes the following new features:
◦
Validation of API calls over HTTPS connection
◦
Generate XML request and their response
◦
Create sample codes in Java and Perl for any API
•
apitest support for Data ONTAP 8.1 Cluster-Mode
•
Added libnetappmgmt.so for the Linux platform
•
Removed support for platforms Red Hat Enterprise Linux 3.0 and Solaris 5.8 and 5.9
