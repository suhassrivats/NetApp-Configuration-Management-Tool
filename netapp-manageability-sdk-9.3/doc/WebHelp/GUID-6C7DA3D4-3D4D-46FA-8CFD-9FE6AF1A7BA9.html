SDK 4.0
This page contains information about what is new in this release.
New in SDK 4.0
The following are new in NetApp Manageability SDK 4.0:
•
NetApp Manageability SDK 4.0 includes Data ONTAP APIs and DataFabric Manager APIs
•
Support for Data ONTAP 7.3.3 APIs
•
Support for DataFabric Manager server 4.0 APIs
•
Support for DataFabric Manager Web services APIs
•
Support for IBM-rebranded Operations Manager, Protection Manager, and Provisioning Manager
•
Sample codes for DataFabric Manager Web services APIs
