Design Guides
Using the design guides, you can use a set of ONTAP APIs and OnCommand Unified Manager APIs, and develop applications for a particular solution domain. The following design guides are available for download at the NetApp Technology Network* site:
•
Performance Management provides documentation on how to use different Performance APIs to get the required data for analyzing the performance of NetApp storage systems.
•
Block Access Management provides information about how to use NetApp open interfaces to manage the block access protocols, such as iSCSI and FCP, and block access entities, such as LUNs, initiator groups, and portsets for NetApp storage systems.
•
Unified Capacity Management provides details of how to use NetApp open interfaces for Unified Capacity management.
•
DataFabric Manager server as a proxy for Storage Management provides information about how to setup Operations Manager to use it as a proxy to communicate with NetApp storage systems.
•
Storage Provisioning Integration with Orchestration Software provides information about application integration with any third-party management tool, orchestration tool, or customer-created tools.
