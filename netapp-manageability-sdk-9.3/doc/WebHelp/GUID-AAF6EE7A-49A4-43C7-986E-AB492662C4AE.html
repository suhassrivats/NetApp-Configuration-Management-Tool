Sample Codes
Core APIs are used to make API calls to a NetApp storage system, a DataFabric Manager server, or an IBM N series system.
The Core API sample codes demonstrate setting up the network connections, server style, and transport type, invoking commands to the server, and retrieving data. The following list describes different sample codes, and check the src/sample/ file for the sample code.
