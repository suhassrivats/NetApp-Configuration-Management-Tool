Sample Codes
This section provides sample codes for the ONTAP APIs, DataFabric Manager server APIs and the DataFabric Manager web services APIs for the OnCommand Unified Manager Core Package, and miscellaneous sample codes.
This section contains
•
The ONTAP sample codes demonstrate how to use the various NetApp storage system features, such as qtrees, quotas, volumes, LUNs, Snapshot copy, SnapMirror technology.
•
The OnCommand Unified Manager sample codes demonstrate how to retrieve information about the DataFabric Manager server, dataset management, DataFabric Manager schedule management, creating and managing provisioning and protection policy, and so on.
•
The OnCommand Unified Manager web services sample codes demonstrate how to retrieve information about the DataFabric Manager server version, vFiler units management, chargeback metrics collections, and so on.
•
The Utility sample codes demonstrate the use of operating system API calls.
