SDK 3.5.1
This page contains information about what is new in this release.
New in SDK 3.5.1
The following are new in Manage ONTAP SDK 3.5.1:
•
Microsoft .NET support for PowerShell 1.0.
•
SDK core APIs for connection timeout in Java and Perl.
•
The Manage ONTAP SDK 3.5.1 documents the APIs of Data ONTAP 8.0 (7-mode).
