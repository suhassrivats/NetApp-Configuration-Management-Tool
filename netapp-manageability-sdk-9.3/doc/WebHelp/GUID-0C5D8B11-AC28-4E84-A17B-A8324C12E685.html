Upgrading the libxml library
NetApp Manageability SDK provides the libxml library. If you want to upgrade the libxml library version in Linux and Windows platforms (32 bit or 64 bit), you can download the libxml source code and build the libraries for your applications.
Prerequisites
You must have downloaded the latest version of libxml library and extracted it into a folder.
•
The following are the steps to download and extract the latest version of libxml library:
1.
Download libxml tarball from the following location:
http://xmlsoft.org/sources/libxml2-x.x.x.tar.gz
2.
Find the latest version of libxml and save the .tar.gz file.
3.
Untar the following files: gunzip libxml2-x.x.x.tar.gz and tar xvf libxml2-x.x.x.tar.
4.
Verify that a subdirectory named libxml2-x.x.x is created.
•
For Windows platform, you must have installed Visual Studio.
Building libxml library in a Linux platform for 32 bit and 64 bit
The following are the steps to build libxml library in Linux platform for 32 bit and 64 bit:
1.
Change directory: $ cd libxml2-x.x.x
2.
Configure libxml: $ ./configure
3.
Generate a libxml library: $ make $ make install
4.
Verify that the location of the libxml library is libxml2-x.x.x/.libs.
5.
Replace the old libxml library with the new libxml library.
Building libxml library in a Windows platform for 32 bit and 64 bit
The following are the steps to build libxml library in Windows platform for 32 bit and 64 bit:
1.
From the Visual Studio command prompt, navigate to the libxml extract folder.
2.
Open the configure.js file in the libxml2-2.9./win32 directory.
3.
Go to line number 42 and update the withIconv field from true to false.
4.
Open the Visual studio command prompt based on your requirement and run as administrator:
◦
For 32-bit platform:
VS2012 x86 Native Tools Command
◦
For 64-bit platform:
VS2012 x64 Cross Tools Command
5.
Create an nmake file called Makefile.msvc: libxml2-x.x.x\win32>cscript configure.js compiler=msvc prefix=win32outLIB include=C:\xml\xml2\libxml2-x.x.x\include\libxml debug=no
6.
Set the path to pick the vcvarsall.bat file: set PATH=C:\Program Files (x86)\Microsoft Visual Studio 11.0\VC;%PATH%
7.
Set the path to pick nmake file and compiler: set PATH=C:\Program Files (x86)\Microsoft Visual Studio 11.0\VC\bin;%PATH%
8.
Execute the following command based on your Windows operating system:
◦
For 32-bit platform: libxml2-x.x.x\win32>vcvarsall.bat x86
◦
For 64-bit platform :libxml2-x.x.x\win32>vcvarsall.bat x64
9.
Generate a libxml library: libxml2-x.x.x\win32>nmake libxml2-x.x.x\win32>name install
10.
Open the library from the libxml2-2.9./win32/win32outLIB path.
11.
Replace the old libxml library with the new libxml library.
