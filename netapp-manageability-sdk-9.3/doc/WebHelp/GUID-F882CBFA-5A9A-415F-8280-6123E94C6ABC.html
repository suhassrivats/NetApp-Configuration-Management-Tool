NetApp Manageability SDK 5.2
NetApp Manageability SDK 5.2 includes the following new features:
•
Support for Perl API Bindings
•
Support for the following operating systems:
◦
Windows Server x64 2012
◦
Oracle Enterprise Linux (32-bit and 64-bit) 6.3
◦
SUSE Linux (32-bit and 64-bit) 11 SP2
◦
IBM AIX (32-bit and 64-bit) 7.1
◦
VMware ESX server 4.1 and VMware ESXi server 5.1
•
Support for the following programming languages:
◦
Python 2.7.3 and 3.3.0
◦
Java 7.0
◦
Ruby 1.9.3 and 2.0
◦
Perl 5.16
•
Support for Microsoft Visual Studio 2010 for C and C++
•
Upgrade for OpenSSL 1.0.1e for C and C++
•
Support for the following browsers:
◦
Google Chrome 30
◦
Mozilla Firefox 23, 24, and 25
◦
Internet Explorer 8, 9, and 10
New Core APIs
The following APIs are added:
C/C++
na_server_set_application_name
na_server_get_application_name
Perl
NaServer::set_application_name
NaServer::get_application_name
NaServer::set_bindings_validation
(only for Perl API bindings)
NaServer::set_bindings_family
(only for Perl API bindings)
Python
NaServer.set_application_name
NaServer.get_application_name
Ruby
NaServer::set_application_name
NaServer::get_application_name
