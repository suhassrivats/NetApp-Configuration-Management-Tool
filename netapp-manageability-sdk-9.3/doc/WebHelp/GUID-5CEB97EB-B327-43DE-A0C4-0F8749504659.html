Perl API bindings
Overview
NetApp Manageability SDK provides support for Perl API bindings for both ONTAP APIs and OnCommand Unified Manager APIs. The Perl API bindings libraries contain interfaces to establish a connection with either the ONTAP server or the OnCommand Unified Manager server. By using these libraries, you can create Perl applications to access and manage the ONTAP server or OnCommand Unified Manager server.
Runtime library
NetApp Manageability SDK Perl API bindings provide a runtime library NaServer.pm, which is available at <installation_folder >/lib/perl/NetApp. This library file enables you to establish a server connection, send requests and receive responses, and interpret error messages. Each binding can be called as a subroutine of NaServer module which in turn invokes the corresponding ONTAP or OnCommand Unified Manager API.
API Definitions
The Perl API bindings contain API definitions for ONTAP and OnCommand Unified Manager. The Perl modules specify textual description of APIs, input, output, and typedef parameters corresponding to these APIs. These modules in turn are used for validating bindings and their input. The definitions are specified as hashes of hashes.
