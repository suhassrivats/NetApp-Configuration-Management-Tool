SDK 3.0
This page contains information about what is new in this release.
New in SDK 3.0R1
This release change is the removal of IDEA and RC5 algorithms from SSL libraries. The IDEA and RC5 algorithms are not supported by Data ONTAP for secure communication.
New in SDK 3.0
Dynamic link libraries are available for C and C++. The Manage ONTAP SDK dynamic link libraries are supported on the following platforms:
•
Windows
•
Linux
•
Solaris
•
HP-UX
•
AIX
The dynamic link libraries are present at the platform-specific library folders under the directory lib.
Manage ONTAP SDK libraries are available on VMware ESX and Oracle Enterprise Linux platforms. You can use the libraries present at lib/linux on these platforms.
