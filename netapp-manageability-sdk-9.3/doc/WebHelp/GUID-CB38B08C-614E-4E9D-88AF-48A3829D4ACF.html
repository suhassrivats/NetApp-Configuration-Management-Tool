SDK 1.3
This page contains information about what is new in this release and the changes that occurred in this release.
New in this release
Following are new in Manage ONTAP SDK 1.3:
•
SSL support
A new core SDK function, na_server_set_transport_type(), configures the ONTAPI server to use either plain HTTP or HTTPS.
•
New Java routines
Two new routines, close() and setKeepAliveEnabled(), have been added to the Java core routine package.
•
FPolicy sample code enhancements
The sample code and documentation for FPolicy have been enhanced.
•
The Manage ONTAP SDK 1.5 documents the API of Data ONTAP 7.1.
