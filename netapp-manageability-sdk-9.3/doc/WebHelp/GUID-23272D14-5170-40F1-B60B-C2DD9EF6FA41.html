NetApp Manageability SDK 5.6
NetApp Manageability SDK 5.6 includes the following new features:
•
Perl API Bindings for ONTAP 9.1
•
Perl API Bindings for OnCommand Unified Manager 7.0
NetApp Manageability SDK 5.6 does not support the following feature:
libnetappmgmt.so library is no longer supported.
Note: However, you can use libnetapp.so, libadt.so, and libxml.so together.
