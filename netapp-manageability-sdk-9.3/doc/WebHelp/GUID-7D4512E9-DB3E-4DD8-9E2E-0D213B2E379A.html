Getting started with SDK
Purpose
This section provides information about the configuration details of the SDK such as system requirements and software support, library information for the supported languages, and so on.
This section contains
•

•

•
System and Software Requirements contains information about the platforms and languages that the SDK supports.
•
Library Information contains information about the static and dynamic libraries that should be linked to the client application.
•
Known Issues contains the issues identified in the different SDK releases.
