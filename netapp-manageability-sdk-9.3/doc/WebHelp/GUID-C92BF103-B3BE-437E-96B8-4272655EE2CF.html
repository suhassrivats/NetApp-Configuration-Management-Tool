ONTAP APIs sample codes
You can use ONTAP APIs retrieve information about various NetApp storage system features, such as qtrees, quotas, volumes, LUNs, Snapshot, SnapMirror, and so on.
The following list describes the sample codes that demonstrate the use of a few ONTAP APIs. Extract the .zip files and check the src/sample/ file for the sample code.
