OnCommand Unified Manager web services sample codes
You can use the Web services APIs for OnCommand Unified Manager Core Package to retrieve information about the DataFabric Manager server version, vFiler units management, chargeback metrics collections, and so on.
The following list describes sample codes that demonstrate the use of a few Web services APIs. Extract the .zip files and check the src/sample/ file for the sample code.
