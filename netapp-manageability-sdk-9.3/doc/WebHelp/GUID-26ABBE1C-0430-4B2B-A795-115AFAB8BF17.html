NetApp Manageability SDK 5.0
The following are new in NetApp Manageability SDK 5.0:
•
The Data ONTAP APIs and the DataFabric Manager server APIs for the OnCommand Core Package are available separately at the NetApp Technology Network site.
•
ZExplore is enhanced and is called ZExplore Development Interface (ZEDI) to test APIs, create sample codes, and create workflows in all the languages supported by the SDK. The following are the new features of ZEDI:
◦
Option to use multiple APIs to build a workflow of APIs with automatically generated raw XML and sample codes
◦
Option to save the raw XML and sample codes
◦
Sample codes in all languages that the SDK supports
◦
Provide sample codes for commonly used workflows
◦
Support on Linux
•
Support for Java API Bindings for both Data ONTAP APIs and DataFabric Manager server APIs for the OnCommand Core Package
•
Support for TLSv1 for Java SDK
•
Support for the following Python versions:
◦
Python 2.4.3 and Python 2.4.4
◦
Python 2.5
◦
Python 2.6.2 and Python 2.6.6
•
Support for HP-UX 64-bit IA64 B.11.31 and later
•
Support for the following operating system versions:
◦
Red Hat Linux v6 (32-bit and 64-bit)
◦
VMware ESXi server 5.0
◦
Oracle Enterprise Linux 5.5, 5.6, and 5.7
