NetApp Manageability SDK 5.1
The following are new in NetApp Manageability SDK 5.1:
•
ZEDI user-interface enhancements:
◦
Option to save connection details, load from a saved connection, and delete connection details.
◦
Added API tab includes edit button to edit a node and enter its new parameter name and value.
◦
New right-click options on the Workflow tab and Added APIs tab.
◦
New icons to distinguish between the imported workflows and the workflows that are saved during the current sessions.
◦
Option to invoke multiple APIs in one request or in sequence.
◦
Option to add API name and the time of API invocation in the XML output as comments.
◦
Saved settings for language preference, API invocation options, and workflow path.
◦
Open the last saved workflows and the last selected API version.
•
Support for OpenSSL 1.0.1c version
•
Support for libxml 2.9.0 version
•
Support for the following operating systems:
◦
Windows 64-bit 2008 R2
◦
Oracle Enterprise Linux 5.8, 6.0, 6.1, and 6.2
◦
Solaris SPARC (32-bit and 64-bit) 11.0
◦
FreeBSD (64-bit) 8.0
•
Support for IPv6 on Java, Python, Perl, Ruby, and .NET
•
Support for Google Chrome 21, 22, and 23 versions
•
Support for Mozilla Firefox 15, 16, 17 versions
