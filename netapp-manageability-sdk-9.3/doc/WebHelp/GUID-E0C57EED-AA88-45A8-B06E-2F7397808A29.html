SDK 3.5 and SDK 3.5P1
This page contains information about what is new in this release.
New in SDK 3.5P1
The following is new in this release:
The buffer overrun that occurs in Manage ONTAP SDK 3.5 is resolved in this release of the Manage ONTAP SDK 3.5P1.
Buffer overrun occurs if you invoke the Manage ONTAP SDK 3.5 core APIs na_child_get_int64 and na_child_get_uint64 on a Windows platform.
New in SDK 3.5
The following are supported in Manage ONTAP SDK 3.5:
•
Windows DCE/RPC support for Java
•
Microsoft .NET support for C# and VB.NET
•
AIX 64-bit and Solaris 64-bit platforms
•
Support for IBM N series systems
•
IPv6 support in C and Java libraries of the SDK
•
Windows 2008 platform support
•
The Manage ONTAP SDK 3.5 documents the APIs of Data ONTAP 7.3.1.
