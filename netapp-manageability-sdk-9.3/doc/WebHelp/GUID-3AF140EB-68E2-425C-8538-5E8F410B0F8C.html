SDK 1.6
This page contains information about APIs or API-sets that are newly added in this release.
New in this release
Following are new in Manage ONTAP SDK 1.6:
•
SSL Support in Perl.
Core SDK function set_transport_type() is enhanced to configure ONTAPI server to use either plain HTTP or HTTPS.
HTTPS support uses Net::SSLeay.pm. Net::SSLeay is not part of standard Perl installation. Net::SSLeay.pm needs to be compiled from source Net_SSLeay.pm-1.xx.tar.gz available at http://www.cpan.org. For instructions to compile Net::SSLeay.pm from source, see the readme file in Net_SSLeay.pm-1.xx.tar.gz.
•
ONTAPI libraries supported on the IBM AIX platform.
libadt.lib, libnetapp.lib, and libxml.lib are available on IBM AIX. These libraries are available in the lib/aix directory.
•
ONTAPI libraries supported on 64-bit Windows.
libadt.lib, libnetapp.lib, and libxml.lib are available on Windows x64, ia64. The Windows x64 version of these libraries is available in the lib/nt/x64 directory. The Windows ia64 version of these libraries is available in the lib/nt/ia64 directory.
•
The Manage ONTAP SDK 1.5 documents the API of Data ONTAP 7.1.
