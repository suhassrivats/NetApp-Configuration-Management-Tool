What the heck is this stuff?

           

This directory (src/sample/C/util) contains common code used (or usable) by more than one sample code application.

na_server.c A C++ class that encapsulates the na_server_* XML manipulation routines
netapp.h Declarations for the C(++) files herein
ontapiver.c Sample code for dealing with ONTAPI versions
na_snapshot.c A C++ class that encapsulates the snapshot-* APIs
